AI Milk Navigator Support Site

1. App Overview

This app includes the following features:

1.1 Milk

• Calculate and display predicted milk intake values.
• Record milk intake amounts.

1.2 Records

• Display milk intake records in a graph.

1.3 Settings

• Input and record child information.
• Display app version information.

2. How to Use

• Input child information from the "Settings" menu.
• Record milk intake amount and time from the "Milk" menu.
• The predicted milk intake value for the next feeding will be displayed in the "Milk" menu.
