1. App Overview

This app includes the following features:

1.1 Book Search

• From the "Book Search" menu, you can search for books by title or author name.
• From the search results, you can search for the books in your favorite library and nearby libraries.
• By adding a book to favorites, you can quickly search for it from the "Favorites" menu.

1.2 Library Search

• From the "Library Search" menu, you can search for libraries on the map.
• From the library search results, you can view detailed information about the libraries.
• By adding a library to favorites, it will be displayed at the top of the search results in the "Book Search" menu.

1.3 Favorites

• From the "Favorites" menu, you can edit favorite information for books and libraries.
• You can register up to 5 books and 1 library.

1.4 Settings

• From the "Settings" menu, you can configure various search conditions for book and library searches.

2. How to Use

• Please search for books from the "Book Search" menu. It will return statuses such as "Available", "Not Available", or "On Loan" for each library.
• Please search for libraries from the "Library Search" menu and add them to favorites. Search results from favorite libraries will be displayed at the top in the book search.
